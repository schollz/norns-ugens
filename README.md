# norns ugens

forked from https://github.com/monome/norns, commit 470031df

## install

```
git clone https://github.com/schollz/norns-ugens
cd norns-ugens/SoftCutHead
make install
```

## run

open `norns-ugens/SoftCutHead/test.scd` in SuperCollider
